基于3D打印的岩心润湿性精细调控及驱油机理研究

第一章 绪论

1.1 研究背景与意义

  • 1.1.1 世界能源需求与提高石油采收率（EOR）的战略意义
  • 1.1.2 润湿性对储层多孔介质微观渗流机制的关键影响
  • 1.1.3 传统实验方法的局限性：天然岩心非均质性与人工岩心润湿单一性
  • 1.1.4 数字岩心与3D打印技术：从“数字重构”到“物理再现”的范式转变

1.2 国内外研究现状

  • 1.2.1 3D打印光敏树脂功能化改性研究进展
    • （1）光敏树脂的流变改性与增强增韧
    • （2）3D打印材料润湿性调控技术研究现状
  • 1.2.2 高精度多材料光固化3D打印工艺研究
    • （1）面曝光（DLP/LCD）成型精度控制方法（光化/物理/算法）
    • （2）多材料切换机制与界面结合性能研究
  • 1.2.3 基于微观模型的驱油机理实验研究进展
    • （1）微流控芯片在渗流力学中的应用
    • （2）3D打印岩心模型在EOR机理研究中的应用现状
  • 1.2.4 现有研究存在的问题与不足
    • （痛点分析：现有打印材料润湿性范围窄、多材料打印精度差、缺乏从矿物学角度的仿生设计。

第三章 面向基体树脂（TMPTA）的高精度打印模型构建与工艺调控

3.1 引言

本章选取TMPTA为模型材料，深度开发Z轴化学控制与XY轴算法控制策略，建立模型并推广至3种典型组分。

3.2 光固化成型理论模型的建立与全系参数预测

  • 3.2.1 光学系统能量分布模型（光机385nm波段高斯分布标定）
  • 3.2.2 典型组分（Type I, III, V）的Beer-Lambert模型标定
    • （仅测试“两头+中间”三点，获取 $C_d$ 和 $E_c$。）
  • 3.2.3 基于组分浓度的五元体系工艺参数预测模型
    • （通过插值计算 Type II/IV 的固化参数，完成全系标定。）

3.3 基于光谱匹配的Z轴固化精度（层厚）控制

  • 3.3.1 Z轴过固化现象与光吸收剂筛选策略
  • 3.3.2 T171光吸收剂与UV色浆的光谱匹配实验（基于TMPTA）
  • 3.3.3 黄色色浆对385nm波段的高效吸收与最佳浓度确定
    • （在基体树脂中确定最佳浓度，解决深层固化问题。）
  • 3.3.4 精度控制策略在五元体系中的验证
    • （验证全系添加同浓度色浆后，Z轴精度均受控。）

3.4 基于计算光刻算法的XY轴固化精度（形状）控制

  • 3.4.1 基体树脂固化形状的正向预测模型、固化能量标定
    • （结合切片图像与光场模型，预测实际固化轮廓。）
  • 3.4.2 基于梯度优化与反卷积算法的切片补偿策略
    • （利用算法逆向生成最优掩模图像，抵消光扩散效应。）
  • 3.4.3 基于IoU（交并比）评价指标的形状精度验证
    • （证明算法对基体材料有效，并推广至全系。）

3.5 多材料打印界面结合与最小特征尺寸测定

  • 3.5.1 多材料微流道界面结合强度测试
  • 3.5.2 优化工艺下五元材料体系的最小打印特征尺寸
    • （测定系统物理极限，作为第四章岩心设计的约束条件。）
